- **`skill-agent-factory`** — Generate component-specific coder agents from project brain and approved component scopes.

- **`skill-project-brain`** — Manage reusable project brain files so agents can resume work across conversations without rescanning the whole repository.
- **`skill-project-onboarding`** — Build the initial project brain by scanning architecture, services, stack, dependencies, environment, and test policy.
- **`skill-qc-handoff`** — Create the mandatory Dev-to-QC handoff document after Code Done.

- **`skill-workflow-policy`** — Validate workflow state transitions, required artifacts, approval gates, and agent permissions.
