# Codex Prompts (generated)

Generated by `scripts/build-codex-prompts.py` from `.claude/commands/`. Codex has no plugin
marketplace; these are user-level custom prompts.

## Install (global, all Codex projects)

```bash
mkdir -p ~/.codex/prompts && cp .codex/prompts/*.md ~/.codex/prompts/
```

Then in the Codex TUI: `/coord`, `/analyze-task`, `/dev`, `/qc`, … become available.

These mirror the command entrypoints; the full stateful workflow still needs the
maestro repo + `AGENTS.md`. Do not edit these files by hand — rerun the generator.
