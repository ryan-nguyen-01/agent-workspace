# Deprecations & Removals

## Rails 8.1

- **Built-in `sidekiq` adapter deprecated** — now provided by the `sidekiq` gem itself
- **`ActiveSupport::Configurable` deprecated**
- **`String#mb_chars` and `ActiveSupport::Multibyte::Chars` deprecated**
- **Active Storage Azure backend removed**
- **`#first`/`#last` without explicit `order` deprecated** — see [active-record-queries.md](active-record-queries.md) for details
