# Workspace

Your local working area. Copy a template here to try it, build scratch projects, or keep
work-in-progress — everything in this folder except this README is **gitignored** (it never pollutes
the maestro repo).

```bash
# try a template locally
cp -R ../templates/sdlc my-app && cd my-app && claude
```

For real projects, copy a template OUT of the repo instead (see ../README.md).
